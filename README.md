### № 01 — LUMIÈRE (фотограф / видеограф)

Премиальный лендинг авторской фото- и видеостудии. Тёмная кинематографичная
база с акцентом «шампань-золото», дисплейный serif Fraunces, монохромная
галерея, оживающая в цвете при наведении. Кастомный курсор, спокойная
кинематографичная анимация.

**Функционал:**
- Портфолио по направлениям — фильтр (свадьбы / портрет / коммерция / lifestyle / видео) с broken-grid галереей и hover-раскрытием подписи
- Пакеты съёмок — переключатель «фото / видео» с анимированным pill и сменой тарифов
- Прайс-лист доп. услуг
- Подготовка к съёмке — пошаговый процесс + чек-лист «как подготовиться»
- Свободные даты — интерактивный календарь (свободно / занято / прошедшие дни, навигация по месяцам); выбранная дата подставляется в форму
- Отзывы — карусель с навигацией
- Заявка на бронь — форма с валидацией (телефон/ник, тип съёмки), состояниями ошибки и экраном успеха
- Адаптив, мобильное меню, scroll-reveal, счётчики, no-JS fallback

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

### № 20 — ШЁРСТКА (груминг-салон)

Тёплый бутик-лендинг груминг-студии для собак и кошек. Светлая палитра
«warm sage» с глубоким хвойно-зелёным акцентом и дозированным тёплым абрикосом,
дисплейный шрифт Rubik (полная кириллица), спокойные анимации.

**Функционал:**
- 🐾 Услуги и прайс по типу питомца — интерактивные табы (мелкие / средние / крупные / длинношёрстные / кошки) с пересчётом цен
- 🖼️ Фото «до / после» — перетаскиваемый слайдер-сравнение (pointer + touch)
- 👩‍🔧 Карточки мастеров со специализацией, опытом и рейтингом
- ➕ Допуход — выбор надбавок с живым подсчётом суммы спа-дня
- 🏆 Программа лояльности «Хвостоклуб»: карта штампов (6-я стрижка в подарок) + три уровня
- 📅 Календарь записи: мастер → дата (ближайшие 14 дней, выходной по вс) → свободные слоты
- ✅ Форма записи с жёсткой валидацией — при пустом/некорректном поле заявка не отправляется (подсветка, текст ошибки, скролл к первой ошибке), маска телефона
- 📱 Адаптив, мобильное меню, scroll-reveal, счётчики, FAQ-аккордеон

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

### № 14 — ВЛАСОВ (инфобиз-эксперт / личный бренд)

Лендинг личного бренда инфобиз-эксперта (наставник по онлайн-запускам).
Намеренно уводит жанр от «инфоцыганского» слопа в сторону серьёзного
editorial-бренда: тёплая светлая бумага, чернильный ink, один смелый акцент
вермилион, дисплейный гротеск Onest, асимметричная сетка.

**Функционал:**
- 🧑‍🏫 Hero личного бренда — сплит с портретом, floating-пруфом и счётчиками результатов
- 🧩 Продукты — bento из трёх форматов (курс-флагман, интенсив, личное наставничество)
- 📈 Кейсы учеников — горизонтальный scroll-snap с метриками «было → стало»
- 🗂️ Программа курса — редакционный индекс из 8 модулей со sticky-интро
- 💳 Тарифы — 3 уровня с переключателем «полная оплата / рассрочка 12 мес» и пересчётом
- 🎁 Вебинар-лид-магнит — отдельный accent-блок с мини-формой регистрации (валидация + экран успеха)
- 💬 Отзывы — masonry-сетка цитат, FAQ-аккордеон на нативных `<details>`
- ✅ Заявка на обучение — две формы с полной валидацией: при пустом/некорректном поле заявка **не отправляется** (подсветка, текст ошибки под полем, фокус на первую ошибку), маска телефона, проверка email и чек-бокса согласия
- 📱 Адаптив, мобильное меню, scroll-reveal, счётчики, магнитные кнопки, prefers-reduced-motion

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

### № 03 — ФОРМА (архитектурное бюро · ИЖС)

Лендинг архитектурного бюро по проектированию частных домов. Намеренно «чертёжный»
визуальный язык: светлая «бумага архитектора», графитовый ink, единственный
сигнальный кобальт (blueprint blue), монументальный гротеск Archivo Expanded и
техническая моноширинная IBM Plex Mono. Видимая сетка, размерные линии, маркеры
листов A-01…A-08 — страница ведёт себя как комплект рабочих чертежей.

**Функционал:**
- 🏗️ Два пути к дому — типовые проекты vs индивидуальное проектирование
- 🧱 Каталог типовых проектов с фильтром по площади (до 120 / 120–180 / от 180 м²)
- 🧮 Калькулятор стоимости проектирования — площадь (слайдер), тип, этажность, класс отделки и доп. разделы; живой пересчёт цены, ставки ₽/м², срока, числа разделов и ориентировочной стоимости стройки
- 📐 Примеры планировок — переключаемые SVG-планы (3 варианта) с легендой: площади, спальни, санузлы, состав помещений
- 📋 Состав документации — аккордеон по разделам проекта (АР, КР, ВК, ОВ, ЭОМ, ПЗ)
- 🗺️ Этапы работы — от консультации до авторского надзора
- ✅ Заявка на бесплатную консультацию — валидация (имя, телефон с маской `+7 (___) ___-__-__`), подсветка ошибок, фокус на первое некорректное поле, экран успеха
- 📱 Адаптив, мобильное меню, scroll-reveal, счётчики, prefers-reduced-motion, fallback для фото

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

### № 16 — MELLOW (кофейня и обжарка · ретро 70-х)

Лендинг ламповой кофейни с собственной обжаркой и винил-баром по вечерам.
«Модный ретро»: приглушённая тёплая палитра 70-х (крем-бумага, жжёный
терракот, охра, олива), вращающееся солнце-санберст, арочные рамки,
бумажная фактура, sticker-кнопки с жёсткой тенью, наклонный marquee и
ретро-скрипт Yellowtail поверх современного гротеска Bricolage Grotesque.

**Функционал:**
- ☕ Меню по категориям (кофе / не кофе / к кофе) — табы со сменой панелей и «звёздными» позициями
- 🫘 Зёрна с собой — карточки моно-сортов с уровнем обжарки и нотами; переключатель «разово / подписка −15%» с живым пересчётом всех цен
- 🎶 Винил по вечерам — тёмная секция с крутящейся пластинкой и расписанием по дням недели (чт–вс) + «сейчас играет»
- 🕗 Часы работы в реальном времени — статус «открыто / закрыто», подсветка текущего дня, расчёт времени ближайшего открытия
- 🗺️ Блок «где мы» с CSS-картой и анимированным пином
- ✅ Бронь столика — степпер числа гостей, форма с валидацией (имя, телефон/ник, время), подсветка ошибок, скролл к первой ошибке и экран успеха
- 📱 Адаптив, мобильное меню, scroll-reveal, счётчики, prefers-reduced-motion

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

### № 12 — ÉCLAT (косметология · клиника эстетической косметологии)

Лендинг клиники доказательной косметологии. Светлый «clinical-luxe» язык:
фарфоровый беж, тёплый эспрессо-ink, рабочий акцент розе-берри и золото для
документов; элегантный serif Cormorant Garamond поверх чистого Manrope.
Тёмные секции (пакет-«хит», CTA, футер) дают премиальный контраст.

**Функционал:**
- ✨ Процедуры с фильтром-табами (уход за лицом / инъекции / аппаратная / тело) — 8 карточек с длительностью и ценой «от»
- 🪞 Интерактивное «до / после» — перетаскиваемый слайдер-сравнение (pointer + touch + клавиши ←/→) с переключателем трёх реальных кейсов и подменой описания
- 👩‍⚕️ Врачи-косметологи — карточки с опытом и сертификатами; кнопка пресетит выбранного врача в форму записи
- 📦 Пакеты процедур — переключатель «курс / разовая» с живым пересчётом цен и выгоды
- 💷 Прозрачный прайс-лист — таблица с фильтром по категориям, бесплатная первичная консультация
- ⚠️ Противопоказания — честный аккордеон (абсолютные / временные / требуют согласования) с дисклеймером о консультации врача
- 🏅 Лицензии и сертификаты — карточки-документы с лайтбокс-модалом + логотипы препаратов
- ✅ Онлайн-запись — 4-шаговый мастер (услуга → врач → дата/время → контакты) с валидацией каждого шага, маской телефона `+7 (___) ___-__-__`, итоговым резюме и персональным экраном успеха; пустое имя/телефон заявку не отправляет
- 📱 Адаптив, мобильное меню, scroll-reveal, count-up счётчики, prefers-reduced-motion, fallback для фото

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.
