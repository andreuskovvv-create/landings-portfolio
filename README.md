### № 15 — VOLT Studio (фитнес-студия)

Премиальный лендинг фитнес-студии «из 2030». Тёмная база с электрическим
lime-акцентом, дисплейный гротеск, плавные анимации.

**Функционал:**
- ⚡ Интерактивное расписание с переключением по дням недели и счётчиком свободных мест
- 👤 Личный кабинет (модал-дашборд: прогресс, ближайшие занятия, абонемент)
- 💳 Онлайн-оплата — модалка с маской карты, валидацией и экраном успеха
- 📋 Тарифы с переключателем «помесячно / за год» и пересчётом цены
- 🎁 Блок акций для новичков (первая тренировка бесплатно, −40%)
- ✅ Форма пробной тренировки с полной валидацией (подсветка пустых/некорректных полей)
- 📱 Полностью адаптивный, мобильное меню, scroll-reveal анимации, счётчики

**Стек:** HTML5, CSS3 (custom properties, grid), vanilla JS. Без зависимостей и сборки.

## Запуск

Открыть `index.html` в корне (портфолио-хаб) или `fitness-studio/index.html` напрямую.
Локальный сервер не требуется.
